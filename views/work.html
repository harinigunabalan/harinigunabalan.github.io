Feedback Driven Development (FDD) in Cloud Applications
This thesis work involves the design and development of an efficient FDD IDE Plugin (Eclipse and/or a browser based, SAP Web IDE plugin) for cloud Applications. The research focuses on mapping the source code artifacts to the set of operation Data points that are collected by logs/analytics of the cloud Applications.
